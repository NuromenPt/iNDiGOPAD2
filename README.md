# iNDiGOPAD2
Indigo Notepad is a lightweight, high-performance web-based text editor designed for speed, security, and clean organization. It operates entirely in the browser using a custom local database structure to keep your data private and accessible.

Key Features & Options:
​Smart Auto-Archive: Automatically saves your work every few seconds to a local database. It uses a unique tagging system to organize notes without the need for complex folders.

​Time-Stamp Tracking: Every time a note is saved, the app automatically generates a temporal tag (format: DDMMYYYY-HHMM), allowing you to track the exact version history in the note list.

​Flexible Layout (Linebreak): Includes a "Linebreak" toggle in the menu. Switch between Pre-wrap (for standard writing) and Pre (no-wrap with horizontal scroll.

​Full Unicode Support: Native UTF-8 compatibility ensures that emojis, special symbols, and multi-language accents are always preserved.

​Instant date/time insertion for logging.

​Tab Support.

​Search: Built-in "Enter-to-find" search bar for 
quick navigation within texts.

​Data Portability: * Save as File: Download the current note as a .txt file with a timestamped filename.

​Import/Export: Full archive backup and restore capability.

​Custom Themes: A curated palette of 12 vibrant themes (from Fuchsia to Royal Blue) that adapt the entire UI to your preference.

​Safety First: Double-confirmation prompts for all destructive actions to prevent accidental data loss.
